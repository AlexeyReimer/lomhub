

ЛомХаб
О проекте
ЛомХаб — агрегатор пунктов приёма металлолома России.
Пользователь может найти ближайшую приёмку, сравнить цены и выбрать наиболее выгодное
предложение.
Основные возможности
Карта приёмок
Поиск по городу
Поиск по металлу
Сравнение цен
Карточка приёмки
Кабинет владельца
История изменения цен
