## 6

Ошибка сервера
Для всех ошибок используется единый стиль оформления.
Основной UX-принцип
Пользователь должен найти лучшую цену максимум за три действия:
Выбрать город или разрешить геолокацию.
Выбрать металл.
Выбрать приемку и построить маршрут.
Если для выполнения задачи требуется больше действий — интерфейс считается слишком
сложным и должен быть переработан.
